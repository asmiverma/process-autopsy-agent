# Demo Script (60-90 seconds)

## 1. Start State (10 seconds)

- Open the dashboard.
- Show stable workflows and baseline heatmap.
- Highlight that the system is continuously monitoring operations.

## 2. Trigger Disruption (10 seconds)

- Click Break It.
- Explain that controlled workflow disruptions are injected for live recovery demonstration.

## 3. Risk Queue (15 seconds)

- Show active disruptions in the risk priority queue.
- Point out risk score ordering and high-risk visibility.

## 4. Audit Trail (15 seconds)

- Show the live audit feed.
- Walk through DiagnosisAgent, ActionAgent, and AuditAgent records.
- Highlight reasoning and confidence as decision evidence.

## 5. Resolution (20 seconds)

- Show issues transitioning through system action.
- Confirm disruptions are resolved by system action where applicable.
- Show escalation path only when dependency constraints require it.

## 6. Learning Panel (10-15 seconds)

- Open stall learning insights.
- Explain that recurring patterns are captured to improve future intervention quality.

## Closing Line

"The system detects workflow disruptions, selects deterministic corrective actions, records every decision, and continuously improves recovery quality from historical behavior."
